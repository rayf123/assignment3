You will need:
- Node
- npm
- a database called "postgres"
- change the user, password, host fields if they are different at the top of index.js and init.js

To install required packages: npm install
To create table and add initial values: cd/db then node init.js
To run the application: node index.js

* Make sure you have Node and NPM installed
